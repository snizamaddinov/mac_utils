### mac-utils

This repository contains simple utility scripts to automate various tasks on macOS using cronjobs or macOS automation tools.
#### Scripts
1. **`download_cleaner`**  
   Organizes the `Downloads` folder by moving files into subfolders based on their file extensions.
   
2. **`lang_switch`** 
   Switches the input source (keyboard language) without showing a pop-up. 
   Easily customizable to include additional languages by modifying the code.
